# Changelog

All notable changes to this project will be documented in this file.

## [0.5.0] - 2025-01-20

### Added
- Customer model with code, name, type, and group segmentation
- CustomerAddress model with label and geocoding support
- ContactPoint model for multi-channel contacts (phone, email, WhatsApp)
- ExternalIdentity model for provider integrations
- CustomerGroup model with price list support
- CustomerService facade
- Gates validation system
- Contrib modules: identifiers, insights, preferences, manychat
- Admin interface with optional Unfold support
